# Cathfy Rails API

### Prerequesites

Install Ruby 2.6.0

```
  rvm install 2.6.0
```

### Getting Starting

```
  $ git clone https://github.com/acmesquita/cathfy-api.git
  $ cd cathfy-api/
  $ rails s -p 3000
```

### Authentication

This project using [tamplate Knock](https://github.com/acmesquita/blank-api-with-knock).
